## 系统简介

财务报销 Agent 是一套面向中小企业的智能报销管理系统，核心能力：

- **AI 发票识别**：上传发票图片，自动提取金额、日期、类型、商家等字段
- **规则自动校验**：依据企业报销政策实时提示超限、违规
- **多级审批流程**：员工提交 → 主管审批 → 财务审批 → 付款
- **异常预警**：重复发票检测、月度超限预警

## 快速启动

### 前置依赖

确保以下服务已启动（首次启动后会持续运行）：

```bash
# 启动 PostgreSQL + Redis + MinIO
cd ~/Documents/Development/PersonalProjects/expense-agent/deploy
docker compose up -d
```

### 启动后端 API

```bash
cd ~/Documents/Development/PersonalProjects/expense-agent/code
source ../venv/bin/activate
uvicorn api.main:app --reload --port 8000
```

### 启动前端

```bash
cd ~/Documents/Development/PersonalProjects/expense-agent
source venv/bin/activate
streamlit run code/app.py
```

启动后访问：
- 前端界面：http://localhost:8501
- API 文档：http://localhost:8000/docs
- MinIO 控制台：http://localhost:9001（账号：`minioadmin` / `minioadmin`）

## 登录与注册

### 登录

1. 打开前端页面，选择「**登录**」标签
2. 输入**用户名**（如 `员工小李`）或**邮箱**（如 `employee@test.com`）
3. 输入密码，点击「登录」

### 注册新账号

1. 选择「**注册**」标签
2. 填写姓名、邮箱、密码、部门
3. 角色默认为**员工**；如需分配主管/财务/管理员角色，联系管理员在数据库中修改
4. 点击「注册」后即可登录

## 员工操作指南

### 提交报销申请

1. 登录后进入「**提交报销**」页面
2. 填写报销标题（如：4月差旅报销）
3. 上传发票图片（支持 JPG / PNG / PDF）
   - 系统将自动识别：**发票类型、金额、开票日期、商家名称**
   - 识别完成后请核对信息，如有误可手动修改
4. 添加备注说明（可选）
5. 点击「提交申请」

> ⚠️ **注意**：提交前请确认发票日期和金额正确，AI 识别偶有误差，建议人工复核

### 查看我的申请

1. 进入「**我的申请**」页面
2. 可查看所有申请的当前状态：

| 状态 | 说明 |
|------|------|
| 草稿 | 已保存未提交 |
| 待审批 | 等待主管审核 |
| 主管已通过 | 等待财务审核 |
| 财务已通过 | 等待付款 |
| 已驳回 | 审批不通过，查看驳回原因 |
| 已付款 | 报销完成 |

## 主管操作指南

### 审批报销申请

1. 登录主管账号，进入「**审批管理**」页面
2. 查看待审批列表，点击某条申请查看详情
3. 查看发票信息、金额、员工备注
4. 选择操作：
   - **通过**：流转至财务审批
   - **驳回**：填写驳回原因，申请状态更新为「已驳回」

> 主管负责一级审批，重点关注：业务合理性、金额是否超出预算

---

## 财务操作指南

### 二级审批

1. 登录财务账号，进入「**审批管理**」页面
2. 查看「主管已通过」的申请
3. 核验发票合规性、金额与规则是否匹配
4. 通过后进入「财务已通过」状态，等待付款

### 标记付款

1. 对「财务已通过」的申请，操作「标记付款」
2. 申请状态更新为「已付款」，流程结束

### 查看统计

财务角色可在「**统计报表**」页面查看：
- 各部门报销金额汇总
- 各类型报销占比
- 月度趋势图

## 报销规则说明

系统内置以下报销限额（单次上限）：

| 报销类型 | 单次上限 | 月度预警阈值 |
|----------|----------|--------------|
| 餐饮 | ¥500 | ¥3,000 |
| 交通 | ¥2,000 | ¥10,000 |
| 住宿 | ¥800 | ¥5,000 |
| 办公用品 | ¥20,000 | ¥100,000 |
| 其他 | ¥500 | ¥3,000 |

超出单次上限时，系统会给出校验警告；月度累计超过预警阈值时，审批页面会显示异常提示。

## 常见问题

**Q：上传发票后识别结果不准确怎么办？**
A：识别结果均可手动编辑，请直接修改后再提交。尤其是发票日期，请核对年份是否正确。

**Q：申请被驳回后可以重新提交吗？**
A：目前需重新发起申请。驳回时主管/财务会填写原因，请根据原因修改后重新提交。

**Q：支持哪些发票格式？**
A：支持 JPG、PNG、PDF 格式。建议使用清晰的扫描件或原图，模糊图片会影响识别准确率。

**Q：忘记密码怎么办？**
A：目前版本暂无自助找回功能，请联系管理员在数据库中重置密码。

**Q：登录时提示「用户名/邮箱或密码错误」？**
A：用户名区分中英文，请确认输入正确（如 `员工小李` 而非 `employee`）。邮箱登录更稳定，推荐使用邮箱。
